# SPLISUM Example Data

This folder contains a small example subset for testing the SPLISUM workflow.

Files:
- `data/sample_library_positive.mgf`: small positive-mode target library subset
- `data/sample_query_positive.mgf`: small positive-mode query subset
- `data/sample_library_negative.mgf`: small negative-mode target library subset
- `data/sample_query_negative.mgf`: small negative-mode query subset
- `data/sample_target_metadata.csv`: metadata extracted from the sample target libraries
- `run_example_positive.sh`: example command for positive mode
- `run_example_negative.sh`: example command for negative mode

The sample files are intentionally small so users can quickly test the package structure and command-line workflow.
